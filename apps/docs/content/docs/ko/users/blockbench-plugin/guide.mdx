이 문서는 작업 지시를 바로 실행하기 전에 전체 제작 흐름을 이해하기 위한 가이드입니다.

## 권장 흐름

1. Blockbench Plugin 설치를 완료합니다.
2. Blockbench Plugin 시작 점검을 완료합니다.
3. 구조 모델링을 먼저 확정하고 텍스처 디테일을 쌓습니다.
4. 구조와 표면 의도가 안정된 뒤 애니메이션을 적용합니다.
5. 내보내기 전에 검증과 프리뷰를 확인합니다.
