## 빠른 요약
기존 애니메이션 클립 속성을 수정합니다.

## 개요
- 도구 이름: `update_animation_clip`
- 카테고리: 애니메이션
- Strict schema: `true`

## 도구 설명
기존 애니메이션 클립 속성을 수정합니다.

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "additionalProperties": false,
  "properties": {
    "id": {
      "type": "string"
    },
    "name": {
      "type": "string"
    },
    "newName": {
      "type": "string"
    },
    "length": {
      "type": "number"
    },
    "loop": {
      "type": "boolean"
    },
    "fps": {
      "type": "number"
    },
    "ifRevision": {
      "type": "string"
    },
    "includeState": {
      "type": "boolean"
    },
    "includeDiff": {
      "type": "boolean"
    },
    "diffDetail": {
      "type": "string",
      "enum": [
        "summary",
        "full"
      ]
    }
  }
}
```

## 필드 레퍼런스
| Field | Type | 필수 | 설명 | 제약 |
| --- | --- | --- | --- | --- |
| `id` | `string` | ❌ | 작업 대상의 고유 id를 지정합니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `name` | `string` | ❌ | 매칭에 사용할 대상 이름을 지정합니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `newName` | `string` | ❌ | `newName` 요청 파라미터입니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `length` | `number` | ❌ | `length` 요청 파라미터입니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `loop` | `boolean` | ❌ | `loop` 요청 파라미터입니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `fps` | `number` | ❌ | `fps` 요청 파라미터입니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `ifRevision` | `string` | ❌ | 낙관적 동시성 제어를 위한 리비전 토큰입니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `includeState` | `boolean` | ❌ | 응답에 최신 프로젝트 스냅샷을 포함합니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `includeDiff` | `boolean` | ❌ | 응답에 리비전 diff 데이터를 포함합니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `diffDetail` | `enum("summary", "full")` | ❌ | diff 포함 시 상세 수준을 제어합니다. | enum: "summary", "full" |
