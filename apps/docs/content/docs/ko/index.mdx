Ashfox는 Blockbench 제작 흐름을 위한 MCP 도구를 제공해 모델 작업, 검증, 내보내기 과정을 예측 가능하게 운영할 수 있게 합니다.

## 핵심 가치

- 모델/텍스처/애니메이션 작업을 MCP 명령으로 표준화
- 핸드오프 전 검증 게이트로 품질 리스크 최소화
- 다국어 가이드를 통한 팀 운영 확장성 확보
