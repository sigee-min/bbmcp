## Overview
- Tool name: `set_trigger_keyframes`
- Category: Animation
- Strict schema: `true`

## Tool Description
Sets trigger keyframes (sound/particle/timeline), one key per call.

## Input Schema (JSON)
```json
{
  "type": "object",
  "required": [
    "clip",
    "channel",
    "keys"
  ],
  "additionalProperties": false,
  "properties": {
    "clipId": {
      "type": "string"
    },
    "clip": {
      "type": "string"
    },
    "channel": {
      "type": "string",
      "enum": [
        "sound",
        "particle",
        "timeline"
      ]
    },
    "keys": {
      "type": "array",
      "minItems": 1,
      "maxItems": 1,
      "items": {
        "type": "object",
        "required": [
          "time",
          "value"
        ],
        "additionalProperties": false,
        "properties": {
          "time": {
            "type": "number"
          },
          "value": {}
        }
      }
    },
    "ifRevision": {
      "type": "string"
    },
    "includeState": {
      "type": "boolean"
    },
    "includeDiff": {
      "type": "boolean"
    },
    "diffDetail": {
      "type": "string",
      "enum": [
        "summary",
        "full"
      ]
    }
  }
}
```
