# Validation and Export Flow

## 흐름

1. 검증 실행 (`project.validate`).
2. 이슈 수정 후 재검증 반복.
3. 산출물 내보내기 (`project.export`).
4. 감사용 trace 로그 내보내기 (`project.export_trace_log`).

## UX 노트

- 검증 결과는 심각도/위치 기준으로 그룹화합니다.
- 가능하면 각 결과에 수정 힌트를 함께 제공합니다.
