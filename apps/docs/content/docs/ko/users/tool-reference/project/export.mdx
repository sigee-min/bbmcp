## 개요
- 도구 이름: `export`
- 카테고리: 프로젝트
- Strict schema: `true`

## 도구 설명
Exports the current project to the requested output format (including native codec targets like glTF/OBJ/FBX when available).

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "required": [
    "format",
    "destPath"
  ],
  "additionalProperties": false,
  "properties": {
    "format": {
      "enum": [
        "java_block_item_json",
        "gecko_geo_anim",
        "animated_java",
        "generic_model_json",
        "gltf",
        "native_codec",
        "auto"
      ]
    },
    "codecId": {
      "type": "string"
    },
    "destPath": {
      "type": "string"
    },
    "includeState": {
      "type": "boolean"
    },
    "ifRevision": {
      "type": "string"
    }
  }
}
```

## 필드 레퍼런스
| Field | Type | 필수 | Description | Constraints |
| --- | --- | --- | --- | --- |
| `format` | `enum("java_block_item_json", "gecko_geo_anim", "animated_java", "generic_model_json", "gltf", "native_codec", "auto")` | ✅ | - | enum: "java_block_item_json", "gecko_geo_anim", "animated_java", "generic_model_json", "gltf", "native_codec", "auto" |
| `codecId` | `string` | ❌ | - | - |
| `destPath` | `string` | ✅ | - | - |
| `includeState` | `boolean` | ❌ | - | - |
| `ifRevision` | `string` | ❌ | - | - |

## 최소 요청 예시
```json
{
  "format": "java_block_item_json",
  "destPath": "./out/model.json"
}
```

## 응답 예시
```json
{
  "structuredContent": {
    "path": "value"
  }
}
```

## 운영 노트
- 변경 도구는 `ifRevision` 사용을 권장합니다.
- 스키마에 없는 필드는 보내지 마세요 (strict schema).
- `validate`로 최종 상태를 확인한 후 `export`를 수행하세요.
