## 개요
- 도구 이름: `assign_texture`
- 카테고리: 텍스처
- Strict schema: `true`

## 도구 설명
Binds a texture to cubes/faces (no UV edits).

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "additionalProperties": false,
  "properties": {
    "textureId": {
      "type": "string"
    },
    "textureName": {
      "type": "string"
    },
    "cubeIds": {
      "type": "array",
      "items": {
        "type": "string"
      },
      "description": "Limit to these cube ids. If cubeNames is also provided, both must match."
    },
    "cubeNames": {
      "type": "array",
      "items": {
        "type": "string"
      },
      "description": "Limit to these cube names. If cubeIds is also provided, both must match."
    },
    "faces": {
      "type": "array",
      "minItems": 1,
      "items": {
        "type": "string",
        "enum": [
          "north",
          "south",
          "east",
          "west",
          "up",
          "down"
        ]
      }
    },
    "ifRevision": {
      "type": "string"
    },
    "includeState": {
      "type": "boolean"
    },
    "includeDiff": {
      "type": "boolean"
    },
    "diffDetail": {
      "type": "string",
      "enum": [
        "summary",
        "full"
      ]
    }
  }
}
```

## 필드 레퍼런스
| Field | Type | 필수 | Description | Constraints |
| --- | --- | --- | --- | --- |
| `textureId` | `string` | ❌ | - | - |
| `textureName` | `string` | ❌ | - | - |
| `cubeIds` | `array<string>` | ❌ | Limit to these cube ids. If cubeNames is also provided, both must match. | - |
| `cubeNames` | `array<string>` | ❌ | Limit to these cube names. If cubeIds is also provided, both must match. | - |
| `faces` | `array<enum("north", "south", "east", "west", "up", "down")>` | ❌ | - | minItems: 1 |
| `ifRevision` | `string` | ❌ | - | - |
| `includeState` | `boolean` | ❌ | - | - |
| `includeDiff` | `boolean` | ❌ | - | - |
| `diffDetail` | `enum("summary", "full")` | ❌ | - | enum: "summary", "full" |

## 최소 요청 예시
```json
{
  "textureName": "texture",
  "cubeNames": [
    "cube"
  ]
}
```

## 응답 예시
```json
{
  "structuredContent": {
    "textureName": "value",
    "cubeCount": 0
  }
}
```

## 운영 노트
- 변경 도구는 `ifRevision` 사용을 권장합니다.
- 스키마에 없는 필드는 보내지 마세요 (strict schema).
- `validate`로 최종 상태를 확인한 후 `export`를 수행하세요.
