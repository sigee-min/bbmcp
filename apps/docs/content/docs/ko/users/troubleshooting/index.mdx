문제 해결 속도를 높이려면 증상을 먼저 분류하고 해당 영역 문서로 바로 이동하는 방식이 가장 효과적입니다. 이 섹션은 연결 실패, revision 충돌, 검증/내보내기 오류, 스키마 캐시 문제를 각각 분리해 진단 경로를 명확하게 제공합니다.

호출이 실패했을 때는 실패 시점을 먼저 구분하세요. 도구 실행 전 연결 단계에서 실패했는지, 변경 단계에서 실패했는지, 검증 또는 내보내기 단계에서 실패했는지에 따라 원인 후보가 크게 달라집니다.
