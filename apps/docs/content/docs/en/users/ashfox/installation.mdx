This installation is for teams that run Ashfox services and connect external MCP clients.

## Requirements

- Node.js 22+
- npm 10+

## Bootstrap local stack

```bash
git clone https://github.com/sigee-min/ashfox.git
cd ashfox
npm install
npm run build
npm --workspace @ashfox/web run build
```

## Run core services

```bash
npm run dev:gateway
npm run dev:worker
```

Optional web shell (separate terminal):

```bash
npm --workspace @ashfox/web run dev
```
