# UV 아틀라스 가이드

UV 아틀라스는 클라이언트가 직접 제어하는 기능이 아니라 런타임 내부에서 자동으로 동작하는 안전장치입니다. 텍스처가 존재하는 상태에서 큐브 지오메트리가 크게 바뀌면 자동 재배치와 재투영이 발생할 수 있습니다.

핵심은 이 동작을 예외가 아니라 정상 흐름으로 받아들이는 것입니다. 아틀라스는 겹침 없는 배치를 목표로 하고, 필요한 경우 해상도 확장 또는 밀도 조정을 통해 편집 가능 상태를 유지하려고 시도합니다.

실무에서는 비율/구조 변경을 먼저 충분히 끝낸 뒤 고밀도 디테일 페인팅으로 들어가면 아틀라스 재배치로 인한 재작업을 줄일 수 있습니다.
