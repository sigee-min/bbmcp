# Error and Revision Contracts

## MUST

- 변경 명령에서 리비전 가드를 강제합니다.
- 검증 실패 시 머신 파싱 가능한 결과를 반환합니다.
- 동시성 충돌 오류와 페이로드 오류를 구분 가능하게 유지합니다.

## SHOULD

- 충돌 오류에 다음 액션 힌트를 제공합니다.
- 재시도 시 trace-log 연계 키를 유지합니다.
