## 개요
- 도구 이름: `add_cube`
- 카테고리: 모델링
- Strict schema: `true`

## 도구 설명
Adds a cube to the current project.

## 최소 요청 예시
```json
{
  "name": "cube",
  "from": [
    0,
    0,
    0
  ],
  "to": [
    1,
    1,
    1
  ]
}
```

## 운영 노트
- 변경 도구는 `ifRevision` 사용을 권장합니다.
- 스키마에 없는 필드는 보내지 마세요 (strict schema).
- `validate`로 최종 상태를 확인한 후 `export`를 수행하세요.
