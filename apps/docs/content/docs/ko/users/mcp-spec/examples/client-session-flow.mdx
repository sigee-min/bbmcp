# Client Session Flow

## 흐름

1. capability 조회 (`system.list_capabilities`).
2. 프로젝트 컨텍스트 확보 (`project.ensure_project`).
3. 읽기 전용 명령으로 연결 상태 확인.
4. 리비전 가드가 포함된 첫 변경 명령 실행.

## UX 노트

- capability 목록과 사용 불가 도구를 초기에 노출합니다.
- 네트워크/동시성 오류에 재시도 가이드를 함께 제공합니다.
