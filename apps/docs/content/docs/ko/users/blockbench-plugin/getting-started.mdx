## 목표

플러그인 설치, 엔드포인트 설정, 첫 MCP 호출이 한 세션에서 모두 정상 동작하는지 확인합니다.

## 첫 세션 흐름

1. Blockbench에서 플러그인 번들을 로드합니다.
2. 엔드포인트 경로를 `/mcp`로 설정합니다.
3. `system.list_capabilities`를 호출합니다.
4. `get_project_state` 같은 읽기 호출을 실행합니다.
5. `ifRevision`을 포함한 안전한 변경 호출 후 `validate`를 실행합니다.

이 흐름이 끝까지 성공하면 워크플로우 가이드와 작업 플레이북으로 진행하세요.
