## 핵심 텍스처 도구

- `assign_texture`: 대상 큐브/페이스에 텍스처 바인딩
- `paint_faces`: 면 단위 도형/선/채우기 페인팅
- `read_texture`: 텍스처 이미지와 메타데이터 조회
- `delete_texture`: 프로젝트 텍스처 삭제

## 미리보기와 점검

- 텍스처 수정 후 `render_preview`로 빠르게 결과를 확인합니다.
- 내보내기 전에 `validate`를 실행해 텍스처 사용 문제를 점검합니다.

## 운영 팁

- 큰 변경보다 작은 페인팅 호출을 반복하는 방식이 안전합니다.
- edge case 확인 시 `read_texture`로 실제 상태를 재확인합니다.
