## 빠른 요약
사운드/파티클/타임라인 트리거 키를 설정합니다.

## 개요
- 도구 이름: `set_trigger_keyframes`
- 카테고리: 애니메이션
- Strict schema: `true`

## 도구 설명
사운드/파티클/타임라인 트리거 키를 설정합니다.

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "required": [
    "clip",
    "channel",
    "keys"
  ],
  "additionalProperties": false,
  "properties": {
    "clipId": {
      "type": "string"
    },
    "clip": {
      "type": "string"
    },
    "channel": {
      "type": "string",
      "enum": [
        "sound",
        "particle",
        "timeline"
      ]
    },
    "keys": {
      "type": "array",
      "minItems": 1,
      "maxItems": 1,
      "items": {
        "type": "object",
        "required": [
          "time",
          "value"
        ],
        "additionalProperties": false,
        "properties": {
          "time": {
            "type": "number"
          },
          "value": {}
        }
      }
    },
    "ifRevision": {
      "type": "string"
    },
    "includeState": {
      "type": "boolean"
    },
    "includeDiff": {
      "type": "boolean"
    },
    "diffDetail": {
      "type": "string",
      "enum": [
        "summary",
        "full"
      ]
    }
  }
}
```

## 필드 레퍼런스
| Field | Type | 필수 | 설명 | 제약 |
| --- | --- | --- | --- | --- |
| `clipId` | `string` | ❌ | 작업 대상 애니메이션 클립 식별자입니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `clip` | `string` | ✅ | 작업 대상 애니메이션 클립 이름입니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `channel` | `enum("sound", "particle", "timeline")` | ✅ | `channel` 요청 파라미터입니다. | enum: "sound", "particle", "timeline" |
| `keys` | `array<object>` | ✅ | `keys` 요청 파라미터입니다. | minItems: 1, maxItems: 1 |
| `keys[].time` | `number` | ✅ | `keys[].time` 요청 파라미터입니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `keys[].value` | `any` | ✅ | `keys[].value` 요청 파라미터입니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `ifRevision` | `string` | ❌ | 낙관적 동시성 제어를 위한 리비전 토큰입니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `includeState` | `boolean` | ❌ | 응답에 최신 프로젝트 스냅샷을 포함합니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `includeDiff` | `boolean` | ❌ | 응답에 리비전 diff 데이터를 포함합니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `diffDetail` | `enum("summary", "full")` | ❌ | diff 포함 시 상세 수준을 제어합니다. | enum: "summary", "full" |

## 최소 요청 예시
```json
{
  "clip": "clip",
  "channel": "timeline",
  "keys": [
    {
      "time": 0,
      "value": "marker"
    }
  ]
}
```

## 응답 예시
이 응답은 스키매틱 예시입니다. 실제 값과 선택 필드는 프로젝트 상태 및 런타임 버전에 따라 달라질 수 있습니다.

```json
{
  "structuredContent": {
    "clip": "value",
    "channel": "value"
  }
}
```

## 운영 노트
- 입력 스키마에 선언된 필드만 전송하세요.
- 응답 예시는 스키매틱이므로 선택 필드는 방어적으로 처리하세요.
- 변경 도구에서 지원되는 경우 `ifRevision`을 사용해 충돌을 줄이세요.
