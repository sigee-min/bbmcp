## Overview
- Tool name: `read_texture`
- Category: Texturing
- Strict schema: `true`

## Tool Description
Reads a texture image (dataUri + metadata) or saves a snapshot to .ashfox/tmp.

## Input Schema (JSON)
```json
{
  "type": "object",
  "additionalProperties": false,
  "properties": {
    "id": {
      "type": "string"
    },
    "name": {
      "type": "string"
    },
    "saveToTmp": {
      "type": "boolean"
    },
    "tmpName": {
      "type": "string"
    },
    "tmpPrefix": {
      "type": "string"
    }
  }
}
```
