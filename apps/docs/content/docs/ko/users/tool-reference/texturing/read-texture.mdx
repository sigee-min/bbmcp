## 빠른 요약
텍스처 원본과 메타데이터를 읽어 디버깅 또는 후속 처리에 사용합니다.

## 개요
- 도구 이름: `read_texture`
- 카테고리: 텍스처
- Strict schema: `true`

## 도구 설명
텍스처 원본과 메타데이터를 읽어 디버깅 또는 후속 처리에 사용합니다.

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "additionalProperties": false,
  "properties": {
    "id": {
      "type": "string"
    },
    "name": {
      "type": "string"
    },
    "saveToTmp": {
      "type": "boolean"
    },
    "tmpName": {
      "type": "string"
    },
    "tmpPrefix": {
      "type": "string"
    }
  }
}
```

## 필드 레퍼런스
| Field | Type | 필수 | 설명 | 제약 |
| --- | --- | --- | --- | --- |
| `id` | `string` | ❌ | 작업 대상의 고유 id를 지정합니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `name` | `string` | ❌ | 매칭에 사용할 대상 이름을 지정합니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `saveToTmp` | `boolean` | ❌ | `saveToTmp` 요청 파라미터입니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `tmpName` | `string` | ❌ | `tmpName` 요청 파라미터입니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `tmpPrefix` | `string` | ❌ | `tmpPrefix` 요청 파라미터입니다. | 선언된 타입 외 추가 제약은 없습니다. |

## 최소 요청 예시
```json
{
  "name": "texture"
}
```

## 응답 예시
이 응답은 스키매틱 예시입니다. 실제 값과 선택 필드는 프로젝트 상태 및 런타임 버전에 따라 달라질 수 있습니다.

```json
{
  "structuredContent": {
    "texture": {
      "mimeType": "value",
      "dataUri": "value"
    }
  }
}
```

## 운영 노트
- 입력 스키마에 선언된 필드만 전송하세요.
- 응답 예시는 스키매틱이므로 선택 필드는 방어적으로 처리하세요.
- 이 도구는 읽기 전용이므로 일반적으로 `ifRevision`이 필요하지 않습니다.
