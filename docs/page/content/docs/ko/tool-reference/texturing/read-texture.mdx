## 개요
- 도구 이름: `read_texture`
- 카테고리: 텍스처
- Strict schema: `true`

## 도구 설명
Reads a texture image (dataUri + metadata) or saves a snapshot to .ashfox/tmp.

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "additionalProperties": false,
  "properties": {
    "id": {
      "type": "string"
    },
    "name": {
      "type": "string"
    },
    "saveToTmp": {
      "type": "boolean"
    },
    "tmpName": {
      "type": "string"
    },
    "tmpPrefix": {
      "type": "string"
    }
  }
}
```

## 필드 레퍼런스
| Field | Type | 필수 | Description | Constraints |
| --- | --- | --- | --- | --- |
| `id` | `string` | ❌ | - | - |
| `name` | `string` | ❌ | - | - |
| `saveToTmp` | `boolean` | ❌ | - | - |
| `tmpName` | `string` | ❌ | - | - |
| `tmpPrefix` | `string` | ❌ | - | - |

## 최소 요청 예시
```json
{
  "name": "texture"
}
```

## 응답 예시
```json
{
  "structuredContent": {
    "texture": {
      "mimeType": "value",
      "dataUri": "value"
    }
  }
}
```

## 운영 노트
- 변경 도구는 `ifRevision` 사용을 권장합니다.
- 스키마에 없는 필드는 보내지 마세요 (strict schema).
- `validate`로 최종 상태를 확인한 후 `export`를 수행하세요.
