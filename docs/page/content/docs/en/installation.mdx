## Requirements

- Blockbench Desktop (latest)
- Node.js 22+ (for source builds)
- npm 10+ (for source builds)

## Fast Install (Plugin URL)

In Blockbench Desktop:

1. Open `File > Plugins > Load Plugin from URL`.
2. Paste the URL below.
3. Install / Load the plugin.

```text
https://github.com/sigee-min/ashfox/releases/latest/download/ashfox.js
```

## Build from Source (Optional)

Clone and build:

```bash
git clone https://github.com/sigee-min/ashfox.git
cd ashfox
npm install
npm run build
```

Or if you already have the repository:

```bash
npm install
npm run build
```

Then load `dist/ashfox.js` in Blockbench Plugin Manager.

## Verify MCP Endpoint

Default endpoint:

```text
http://127.0.0.1:8787/mcp
```

Expected result:
- Your MCP client can call `list_capabilities`.
- Response includes tool schemas and `toolRegistry.hash`.

## URL and Runtime Configuration

- Official guide URL: `https://ashfox.sigee.xyz`
- Default MCP endpoint: `http://127.0.0.1:8787/mcp`
- `ASHFOX_HOST`, `ASHFOX_PORT`, `ASHFOX_PATH`: override MCP host/port/path.

Example:

```bash
ASHFOX_HOST=127.0.0.1
ASHFOX_PORT=8787
ASHFOX_PATH=/mcp
```
