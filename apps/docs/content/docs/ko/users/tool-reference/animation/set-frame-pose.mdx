## 개요
- 도구 이름: `set_frame_pose`
- 카테고리: 애니메이션
- Strict schema: `true`

## 도구 설명
Sets a pose frame for multiple bones at a single frame (rot/pos/scale).

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "required": [
    "clip",
    "frame",
    "bones"
  ],
  "additionalProperties": false,
  "properties": {
    "clipId": {
      "type": "string"
    },
    "clip": {
      "type": "string"
    },
    "frame": {
      "type": "number"
    },
    "bones": {
      "type": "array",
      "minItems": 1,
      "items": {
        "type": "object",
        "required": [
          "name"
        ],
        "additionalProperties": false,
        "properties": {
          "name": {
            "type": "string"
          },
          "rot": {
            "type": "array",
            "items": {
              "type": "number"
            },
            "minItems": 3,
            "maxItems": 3
          },
          "pos": {
            "type": "array",
            "items": {
              "type": "number"
            },
            "minItems": 3,
            "maxItems": 3
          },
          "scale": {
            "type": "array",
            "items": {
              "type": "number"
            },
            "minItems": 3,
            "maxItems": 3
          },
          "interp": {
            "type": "string",
            "enum": [
              "linear",
              "step",
              "catmullrom"
            ]
          }
        },
        "anyOf": [
          {
            "required": [
              "rot"
            ]
          },
          {
            "required": [
              "pos"

          },
          {
            "required": [

          }

    },

      "type": "string",

      "type": "boolean"

      "type": "string",

        "summary",
        "full"
      ]

  }
}
```

## 필드 레퍼런스
| Field | Type | 필수 | Description | Constraints |

| `clipId` | `string` | ❌ | - | - |
| `clip` | `string` | ✅ | - | - |
| `frame` | `number` | ✅ | - | - |
| `bones` | `array<any \| any \| any>` | ✅ | - | minItems: 1 |
| `bones[].name` | `string` | ✅ | - | - |
| `bones[].rot` | `array<number>` | ❌ | - | minItems: 3, maxItems: 3 |
| `bones[].pos` | `array<number>` | ❌ | - | minItems: 3, maxItems: 3 |
| `bones[].scale` | `array<number>` | ❌ | - | minItems: 3, maxItems: 3 |
| `bones[].interp` | `enum("linear", "step", "catmullrom")` | ❌ | - | enum: "linear", "step", "catmullrom" |
| `interp` | `enum("linear", "step", "catmullrom")` | ❌ | - | enum: "linear", "step", "catmullrom" |
| `ifRevision` | `string` | ❌ | - | - |
| `includeState` | `boolean` | ❌ | - | - |

## 운영 노트
- 변경 도구는 `ifRevision` 사용을 권장합니다.
- 스키마에 없는 필드는 보내지 마세요 (strict schema).
- `validate`로 최종 상태를 확인한 후 `export`를 수행하세요.
