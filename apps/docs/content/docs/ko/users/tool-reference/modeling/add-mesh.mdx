## 개요
- 도구 이름: `add_mesh`
- 카테고리: 모델링
- Strict schema: `true`

## 도구 설명
Adds a mesh with explicit vertices/faces (mesh-centered modeling).

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "required": [
    "name",
    "vertices",
    "faces"
  ],
  "additionalProperties": false,
  "properties": {
    "id": {
      "type": "string"
    },
    "name": {
      "type": "string"
    },

    },
    "boneId": {
      "type": "string"

    "origin": {

      },
      "minItems": 3,
      "maxItems": 3
```

## 필드 레퍼런스
| Field | Type | 필수 | Description | Constraints |
| --- | --- | --- | --- | --- |
| `id` | `string` | ❌ | - | - |
| `name` | `string` | ✅ | - | - |
| `bone` | `string` | ❌ | - | - |
| `boneId` | `string` | ❌ | - | - |
| `origin` | `array<number>` | ❌ | - | minItems: 3, maxItems: 3 |
| `rotation` | `array<number>` | ❌ | - | minItems: 3, maxItems: 3 |
| `visibility` | `boolean` | ❌ | - | - |
| `uvPolicy` | `object` | ❌ | - | additionalProperties=false |
| `uvPolicy.symmetryAxis` | `enum("none", "x", "y", "z")` | ❌ | - | enum: "none", "x", "y", "z" |
| `uvPolicy.texelDensity` | `number` | ❌ | - | min: 0.25, max: 64 |
| `uvPolicy.padding` | `number` | ❌ | - | min: 0, max: 16 |
| `vertices` | `array<object>` | ✅ | - | minItems: 3 |
| `vertices[].id` | `string` | ✅ | - | - |
| `vertices[].pos` | `array<number>` | ✅ | - | minItems: 3, maxItems: 3 |
| `faces` | `array<object>` | ✅ | - | minItems: 1 |
| `faces[].id` | `string` | ❌ | - | - |
| `faces[].vertices` | `array<string>` | ✅ | - | minItems: 3 |
| `faces[].texture` | `string \| enum(false)` | ❌ | - | anyOf(2) |
| `ifRevision` | `string` | ❌ | - | - |
| `includeState` | `boolean` | ❌ | - | - |
| `includeDiff` | `boolean` | ❌ | - | - |
| `diffDetail` | `enum("summary", "full")` | ❌ | - | enum: "summary", "full" |

## 최소 요청 예시
```json
{
  "name": "mesh",
  "vertices": [
    {
      "id": "v0",
      "pos": [
        0,
        0,
        0
      ]
    },
    {
      "id": "v1",
      "pos": [
        1,
        0,
        0
      ]
    },
    {
      "id": "v2",
      "pos": [
        0,
        1,
        0
      ]
    }
  ],
  "faces": [
    {
      "vertices": [
        "v0",
        "v1",
        "v2"
      ]
    }
  ]
}
```

## 운영 노트
- 변경 도구는 `ifRevision` 사용을 권장합니다.
- 스키마에 없는 필드는 보내지 마세요 (strict schema).
- `validate`로 최종 상태를 확인한 후 `export`를 수행하세요.
