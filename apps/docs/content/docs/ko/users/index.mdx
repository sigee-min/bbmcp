사용자 문서는 실제 사용 환경 기준으로 두 트랙으로 구성됩니다. 먼저 본인 환경에 맞는 트랙을 선택하고, 해당 트랙의 설치와 시작, 워크플로우 가이드를 완료한 뒤 작업 플레이북으로 내려가는 흐름을 권장합니다.

## 권장 읽기 순서

1. `Blockbench Plugin` 또는 `Ashfox` 트랙을 선택합니다.
2. 선택한 트랙의 `설치`를 완료합니다.
3. 선택한 트랙의 `시작`과 `워크플로우 가이드`를 확인합니다.
4. 실제 실행은 트랙별 `작업 플레이북`에서 진행합니다.
5. 상세 스펙은 `도구 레퍼런스`, 장애 대응은 `문제 해결`에서 확인합니다.
