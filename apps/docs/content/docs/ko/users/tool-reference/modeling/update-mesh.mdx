## 빠른 요약
기존 메시의 정점/면/변환 정보를 갱신합니다.

## 개요
- 도구 이름: `update_mesh`
- 카테고리: 모델링
- Strict schema: `true`

## 도구 설명
기존 메시의 정점/면/변환 정보를 갱신합니다.

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "additionalProperties": false,
  "properties": {
    "id": {
      "type": "string"
    },
    "name": {
      "type": "string"
    },
    "newName": {
      "type": "string"
    },
    "bone": {
      "type": "string"
    },
    "boneId": {
      "type": "string"
    },
    "boneRoot": {
      "type": "boolean"
    },
    "origin": {
      "type": "array",
      "items": {
        "type": "number"
      },
      "minItems": 3,
      "maxItems": 3
    },
    "rotation": {
      "type": "array",
      "items": {
        "type": "number"
      },
      "minItems": 3,
      "maxItems": 3
    },
    "visibility": {
      "type": "boolean"
    },
    "uvPolicy": {
      "type": "object",
      "additionalProperties": false,
      "properties": {
        "symmetryAxis": {
          "type": "string",
          "enum": [
            "none",
            "x",
            "y",
            "z"
          ]
        },
        "texelDensity": {
          "type": "number",
          "minimum": 0.25,
          "maximum": 64
        },
        "padding": {
          "type": "number",
          "minimum": 0,
          "maximum": 16
        }
      }
    },
    "vertices": {
      "type": "array",
      "minItems": 3,
      "items": {
        "type": "object",
        "required": [
          "id",
          "pos"
        ],
        "additionalProperties": false,
        "properties": {
          "id": {
            "type": "string"
          },
          "pos": {
            "type": "array",
            "items": {
              "type": "number"
            },
            "minItems": 3,
            "maxItems": 3
          }
        }
      }
    },
    "faces": {
      "type": "array",
      "minItems": 1,
      "items": {
        "type": "object",
        "required": [
          "vertices"
        ],
        "additionalProperties": false,
        "properties": {
          "id": {
            "type": "string"
          },
          "vertices": {
            "type": "array",
            "minItems": 3,
            "items": {
              "type": "string"
            }
          },
          "texture": {
            "anyOf": [
              {
                "type": "string"
              },
              {
                "type": "boolean",
                "enum": [
                  false
                ]
              }
            ]
          }
        }
      }
    },
    "ifRevision": {
      "type": "string"
    },
    "includeState": {
      "type": "boolean"
    },
    "includeDiff": {
      "type": "boolean"
    },
    "diffDetail": {
      "type": "string",
      "enum": [
        "summary",
        "full"
      ]
    }
  }
}
```

## 필드 레퍼런스
| Field | Type | 필수 | 설명 | 제약 |
| --- | --- | --- | --- | --- |
| `id` | `string` | ❌ | 작업 대상의 고유 id를 지정합니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `name` | `string` | ❌ | 매칭에 사용할 대상 이름을 지정합니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `newName` | `string` | ❌ | `newName` 요청 파라미터입니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `bone` | `string` | ❌ | `bone` 요청 파라미터입니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `boneId` | `string` | ❌ | `boneId` 요청 파라미터입니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `boneRoot` | `boolean` | ❌ | `boneRoot` 요청 파라미터입니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `origin` | `array<number>` | ❌ | `origin` 요청 파라미터입니다. | minItems: 3, maxItems: 3 |
| `rotation` | `array<number>` | ❌ | `rotation` 요청 파라미터입니다. | minItems: 3, maxItems: 3 |
| `visibility` | `boolean` | ❌ | `visibility` 요청 파라미터입니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `uvPolicy` | `object` | ❌ | `uvPolicy` 요청 파라미터입니다. | additionalProperties=false |
| `uvPolicy.symmetryAxis` | `enum("none", "x", "y", "z")` | ❌ | `uvPolicy.symmetryAxis` 요청 파라미터입니다. | enum: "none", "x", "y", "z" |
| `uvPolicy.texelDensity` | `number` | ❌ | `uvPolicy.texelDensity` 요청 파라미터입니다. | min: 0.25, max: 64 |
| `uvPolicy.padding` | `number` | ❌ | `uvPolicy.padding` 요청 파라미터입니다. | min: 0, max: 16 |
| `vertices` | `array<object>` | ❌ | `vertices` 요청 파라미터입니다. | minItems: 3 |
| `vertices[].id` | `string` | ✅ | 작업 대상의 고유 id를 지정합니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `vertices[].pos` | `array<number>` | ✅ | `vertices[].pos` 요청 파라미터입니다. | minItems: 3, maxItems: 3 |
| `faces` | `array<object>` | ❌ | `faces` 요청 파라미터입니다. | minItems: 1 |
| `faces[].id` | `string` | ❌ | 작업 대상의 고유 id를 지정합니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `faces[].vertices` | `array<string>` | ✅ | `faces[].vertices` 요청 파라미터입니다. | minItems: 3 |
| `faces[].texture` | `string \ | enum(false)` | ❌ | 선언된 타입 외 추가 제약은 없습니다. |
| `ifRevision` | `string` | ❌ | 낙관적 동시성 제어를 위한 리비전 토큰입니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `includeState` | `boolean` | ❌ | 응답에 최신 프로젝트 스냅샷을 포함합니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `includeDiff` | `boolean` | ❌ | 응답에 리비전 diff 데이터를 포함합니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `diffDetail` | `enum("summary", "full")` | ❌ | diff 포함 시 상세 수준을 제어합니다. | enum: "summary", "full" |

## 최소 요청 예시
```json
{
  "name": "mesh"
}
```

## 응답 예시
이 응답은 스키매틱 예시입니다. 실제 값과 선택 필드는 프로젝트 상태 및 런타임 버전에 따라 달라질 수 있습니다.

```json
{
  "structuredContent": {
    "id": "value",
    "name": "value"
  }
}
```

## 운영 노트
- 입력 스키마에 선언된 필드만 전송하세요.
- 응답 예시는 스키매틱이므로 선택 필드는 방어적으로 처리하세요.
- 변경 도구에서 지원되는 경우 `ifRevision`을 사용해 충돌을 줄이세요.
