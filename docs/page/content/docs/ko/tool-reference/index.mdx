## 핵심 규칙

- 변경 도구 호출에 `ifRevision`을 포함합니다.
- 저수준 단위 변경(명확한 한 단계 수정)을 유지합니다.
- `export` 전에 `validate`를 실행합니다.
- `toolRegistry.hash`가 바뀌면 `list_capabilities` 기반 스키마를 갱신합니다.
