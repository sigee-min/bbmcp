이 문서는 개별 작업 실행 전에 서비스 운영 경계와 점검 기준을 맞추기 위한 가이드입니다.

## 권장 흐름

1. Ashfox 설치를 완료합니다.
2. Ashfox 시작 점검을 완료합니다.
3. gateway/worker 연결성과 capability 동기화를 확인합니다.
4. 운영 루프에 검증과 프리뷰 체크포인트를 포함합니다.
5. 엔드포인트/스키마 이상 시 문제 해결 문서로 전환합니다.
