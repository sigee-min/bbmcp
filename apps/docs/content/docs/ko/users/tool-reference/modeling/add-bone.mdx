## 빠른 요약
새 본을 추가합니다.

## 개요
- 도구 이름: `add_bone`
- 카테고리: 모델링
- Strict schema: `true`

## 도구 설명
새 본을 추가합니다.

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "required": [
    "name"
  ],
  "additionalProperties": false,
  "properties": {
    "id": {
      "type": "string"
    },
    "name": {
      "type": "string"
    },
    "parent": {
      "type": "string"
    },
    "parentId": {
      "type": "string"
    },
    "pivot": {
      "type": "array",
      "items": {
        "type": "number"
      },
      "minItems": 3,
      "maxItems": 3
    },
    "rotation": {
      "type": "array",
      "items": {
        "type": "number"
      },
      "minItems": 3,
      "maxItems": 3
    },
    "scale": {
      "type": "array",
      "items": {
        "type": "number"
      },
      "minItems": 3,
      "maxItems": 3
    },
    "visibility": {
      "type": "boolean"
    },
    "ifRevision": {
      "type": "string"
    },
    "includeState": {
      "type": "boolean"
    },
    "includeDiff": {
      "type": "boolean"
    },
    "diffDetail": {
      "type": "string",
      "enum": [
        "summary",
        "full"
      ]
    }
  }
}
```

## 필드 레퍼런스
| Field | Type | 필수 | 설명 | 제약 |
| --- | --- | --- | --- | --- |
| `id` | `string` | ❌ | 작업 대상의 고유 id를 지정합니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `name` | `string` | ✅ | 매칭에 사용할 대상 이름을 지정합니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `parent` | `string` | ❌ | `parent` 요청 파라미터입니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `parentId` | `string` | ❌ | `parentId` 요청 파라미터입니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `pivot` | `array<number>` | ❌ | `pivot` 요청 파라미터입니다. | minItems: 3, maxItems: 3 |
| `rotation` | `array<number>` | ❌ | `rotation` 요청 파라미터입니다. | minItems: 3, maxItems: 3 |
| `scale` | `array<number>` | ❌ | `scale` 요청 파라미터입니다. | minItems: 3, maxItems: 3 |
| `visibility` | `boolean` | ❌ | `visibility` 요청 파라미터입니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `ifRevision` | `string` | ❌ | 낙관적 동시성 제어를 위한 리비전 토큰입니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `includeState` | `boolean` | ❌ | 응답에 최신 프로젝트 스냅샷을 포함합니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `includeDiff` | `boolean` | ❌ | 응답에 리비전 diff 데이터를 포함합니다. | 선언된 타입 외 추가 제약은 없습니다. |
| `diffDetail` | `enum("summary", "full")` | ❌ | diff 포함 시 상세 수준을 제어합니다. | enum: "summary", "full" |

## 최소 요청 예시
```json
{
  "name": "value"
}
```

## 응답 예시
이 응답은 스키매틱 예시입니다. 실제 값과 선택 필드는 프로젝트 상태 및 런타임 버전에 따라 달라질 수 있습니다.

```json
{
  "structuredContent": {
    "id": "value",
    "name": "value"
  }
}
```

## 운영 노트
- 입력 스키마에 선언된 필드만 전송하세요.
- 응답 예시는 스키매틱이므로 선택 필드는 방어적으로 처리하세요.
- 변경 도구에서 지원되는 경우 `ifRevision`을 사용해 충돌을 줄이세요.
