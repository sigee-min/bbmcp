## 빠른 요약
기존 큐브의 형상/변환/소속 정보를 갱신합니다.

## 개요
- 도구 이름: `update_cube`
- 카테고리: 모델링
- Strict schema: `true`

## 도구 설명
기존 큐브의 형상/변환/소속 정보를 갱신합니다.

## 입력 스키마 (JSON)
```json

  "type": "object",
  "additionalProperties": false,
  "properties": {
    "id": {
      "type": "string"
    },

      "type": "string"

    "bone": {

    "boneId": {
      "type": "string"
    },

      "type": "boolean"
    },
    "from": {
      "type": "array",
      "items": {
        "type": "number"
      },
      "minItems": 3,
      "maxItems": 3
    },
    "to": {
      "type": "array",
      "items": {
        "type": "number"
      },
      "minItems": 3,
      "maxItems": 3
    },
    "origin": {
      "type": "array",
      "items": {
        "type": "number"
      },
      "minItems": 3,
      "maxItems": 3
    },
    "rotation": {
      "type": "array",
      "items": {
        "type": "number"
      },
      "minItems": 3,
      "maxItems": 3
    },
    "inflate": {
      "type": "number"
    },
    "mirror": {
      "type": "boolean"
    },
    "visibility": {
      "type": "boolean"
    },
    "boxUv": {

      "type": "string",

      ]
    }
  }
}
```

| `boneRoot` | `boolean` | ❌ | `boneRoot` 요청 파라미터입니다. | 선언된 타입 외 추가 제약은 없습니다. |

| `boxUv` | `boolean` | ❌ | `boxUv` 요청 파라미터입니다. | 선언된 타입 외 추가 제약은 없습니다. |

## 최소 요청 예시
```json
{
  "name": "cube"
}
```

## 운영 노트
