# bbmcp page

Static docs site built with Next.js + Fumadocs.

## Local Development

```bash
npm install
npm run dev
```

Default localized routes:

- `http://localhost:3000/en`
- `http://localhost:3000/ko`

## Static Build

```bash
npm run build
```

Output directory: `out/`

## Environment Variables

Copy `.env.example` and set values when needed:

- `DOCS_BASE_PATH`: optional base path when serving docs under a subpath
- `DOCS_SITE_URL`: optional canonical URL for metadata (example: `https://docs.example.com/`)

## Content Structure

- `content/docs/en/*`: English docs
- `content/docs/ko/*`: Korean docs
- locale-aware routes are generated by `src/lib/i18n.ts` + `src/lib/source.ts`

## CI

Repository CI runs `docs/page` static build as a validation step (`docs-static-check` job).
