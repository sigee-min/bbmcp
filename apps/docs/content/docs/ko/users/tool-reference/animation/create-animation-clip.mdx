## 개요
- 도구 이름: `create_animation_clip`
- 카테고리: 애니메이션
- Strict schema: `true`

## 도구 설명
Creates an animation clip (low-level).

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "required": [
    "name",
    "length",
    "loop",
    "fps"
  ],
  "additionalProperties": false,
  "properties": {
    "id": {
      "type": "string"
    },
    "name": {
      "type": "string"
    },
    "length": {
      "type": "number"
    },
    "loop": {
      "type": "boolean"
    },
    "fps": {
      "type": "number"
    },
    "ifRevision": {
      "type": "string"
    },
    "includeState": {
      "type": "boolean"
    },
    "includeDiff": {
      "type": "boolean"
    },
    "diffDetail": {
      "type": "string",
      "enum": [
        "summary",
        "full"
      ]
    }
  }
}
```
