This installation is for creators who use Blockbench Desktop directly.

## Requirements

- Blockbench Desktop
- Node.js 22+
- npm 10+

## Quick install (plugin URL)

1. Open `File > Plugins > Load Plugin from URL` in Blockbench Desktop.
2. Paste the URL below and load the plugin.

```text
https://github.com/sigee-min/ashfox/releases/latest/download/ashfox-bbplugin.js
```

## Build from source (optional)

```bash
git clone https://github.com/sigee-min/ashfox.git
cd ashfox
npm install
npm run build:plugin-desktop
```

Then load `dist/ashfox-bbplugin.js` in Blockbench Plugin Manager.
