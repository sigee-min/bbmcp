## 빠른 요약
현재 프로젝트의 구조, 텍스처, 애니메이션 상태를 읽기 전용으로 조회합니다.

## 개요
- 도구 이름: `get_project_state`
- 카테고리: 프로젝트
- Strict schema: `true`

## 도구 설명
현재 프로젝트의 구조, 텍스처, 애니메이션 상태를 읽기 전용으로 조회합니다.

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "additionalProperties": false,
  "properties": {
    "detail": {
      "type": "string",
      "enum": [
        "summary",
        "full"
      ]
    },
    "includeUsage": {
      "type": "boolean",
      "description": "Include textureUsage in the response (defaults to true when detail=full)."
    }
  }
}
```

## 필드 레퍼런스
| Field | Type | 필수 | 설명 | 제약 |
| --- | --- | --- | --- | --- |
| `detail` | `enum("summary", "full")` | ❌ | 응답 상세 수준을 제어합니다. | enum: "summary", "full" |
| `includeUsage` | `boolean` | ❌ | Include textureUsage in the response (defaults to true when detail=full). | 선언된 타입 외 추가 제약은 없습니다. |

## 최소 요청 예시
```json
{}
```

이 응답은 스키매틱 예시입니다. 실제 값과 선택 필드는 프로젝트 상태 및 런타임 버전에 따라 달라질 수 있습니다.

## 운영 노트
- 입력 스키마에 선언된 필드만 전송하세요.
- 응답 예시는 스키매틱이므로 선택 필드는 방어적으로 처리하세요.
- 이 도구는 읽기 전용이므로 일반적으로 `ifRevision`이 필요하지 않습니다.
