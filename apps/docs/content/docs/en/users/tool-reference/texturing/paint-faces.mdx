## Quick Summary
Paints one cube face with one drawing op (UV handled internally). Default coordSpace=face; use coordSpace=texture with width/height for texture-space coordinates.

## Overview
- Tool name: `paint_faces`
- Category: Texturing
- Strict schema: `true`

## Tool Description
Paints one cube face with one drawing op (UV handled internally). Default coordSpace=face; use coordSpace=texture with width/height for texture-space coordinates.
