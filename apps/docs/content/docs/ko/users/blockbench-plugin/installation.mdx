이 설치 문서는 Blockbench Desktop에서 직접 제작하는 사용자를 위한 경로입니다.

## 요구 사항

- Blockbench Desktop
- Node.js 22+
- npm 10+

## 빠른 설치 (플러그인 URL)

1. Blockbench Desktop에서 `File > Plugins > Load Plugin from URL`를 엽니다.
2. 아래 URL을 붙여 넣고 플러그인을 로드합니다.

```text
https://github.com/sigee-min/ashfox/releases/latest/download/ashfox-bbplugin.js
```

## 소스 빌드 설치 (선택)

```bash
git clone https://github.com/sigee-min/ashfox.git
cd ashfox
npm install
npm run build:plugin-desktop
```

빌드 후 Blockbench Plugin Manager에서 `dist/ashfox-bbplugin.js`를 로드합니다.
