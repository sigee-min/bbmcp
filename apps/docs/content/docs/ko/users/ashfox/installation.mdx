이 설치 문서는 Ashfox 서비스를 운영하고 외부 MCP 클라이언트를 연결하는 팀을 위한 경로입니다.

## 요구 사항

- Node.js 22+
- npm 10+

## 로컬 스택 부트스트랩

```bash
git clone https://github.com/sigee-min/ashfox.git
cd ashfox
npm install
npm run build
npm --workspace @ashfox/web run build
```

## 핵심 서비스 실행

```bash
npm run dev:gateway
npm run dev:worker
```

선택 사항 (웹 셸, 별도 터미널):

```bash
npm --workspace @ashfox/web run dev
```
