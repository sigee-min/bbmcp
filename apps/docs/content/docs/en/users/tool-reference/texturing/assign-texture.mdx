## Overview
- Tool name: `assign_texture`
- Category: Texturing
- Strict schema: `true`

## Tool Description
Binds a texture to cubes/faces (no UV edits).

## Input Schema (JSON)
```json
{
  "type": "object",
  "additionalProperties": false,
  "properties": {
    "textureId": {
      "type": "string"
    },
    "textureName": {
      "type": "string"
    },
    "cubeIds": {
      "type": "array",
      "items": {
        "type": "string"
      },
      "description": "Limit to these cube ids. If cubeNames is also provided, both must match."
    },
    "cubeNames": {
      "type": "array",
      "items": {
        "type": "string"
      },
      "description": "Limit to these cube names. If cubeIds is also provided, both must match."
    },
    "faces": {
      "type": "array",
      "minItems": 1,
      "items": {
        "type": "string",
        "enum": [
          "north",
          "south",
          "east",
          "west",
          "up",
          "down"
        ]
      }
    },
    "ifRevision": {
      "type": "string"
    },
    "includeState": {
      "type": "boolean"
    },
    "includeDiff": {
      "type": "boolean"
    },
    "diffDetail": {
      "type": "string",
      "enum": [
        "summary",
        "full"
      ]
    }
  }
}
```
