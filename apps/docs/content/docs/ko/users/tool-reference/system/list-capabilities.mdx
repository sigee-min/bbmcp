## 빠른 요약
런타임에서 사용 가능한 포맷, 제한치, 기능 플래그를 조회합니다.

## 개요
- 도구 이름: `list_capabilities`
- 카테고리: 시스템
- Strict schema: `true`

## 도구 설명
런타임에서 사용 가능한 포맷, 제한치, 기능 플래그를 조회합니다.

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "additionalProperties": false
}
```

## 필드 레퍼런스
입력 필드가 없습니다.

## 최소 요청 예시
```json
{}
```

## 응답 예시
이 응답은 스키매틱 예시입니다. 실제 값과 선택 필드는 프로젝트 상태 및 런타임 버전에 따라 달라질 수 있습니다.

```json
{
  "structuredContent": {
    "pluginVersion": "value",
    "blockbenchVersion": "value",
    "formats": [
      {
        "format": "Java Block/Item",
        "animations": false,
        "enabled": false
      }
    ],
    "limits": {
      "maxCubes": 0,
      "maxTextureSize": 0,
      "maxAnimationSeconds": 0
    }
  }
}
```

## 운영 노트
- 입력 스키마에 선언된 필드만 전송하세요.
- 응답 예시는 스키매틱이므로 선택 필드는 방어적으로 처리하세요.
- 이 도구는 읽기 전용이므로 일반적으로 `ifRevision`이 필요하지 않습니다.
