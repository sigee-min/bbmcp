## Recommended flow

1. Complete Ashfox installation.
2. Complete Ashfox getting started checks.
3. Verify gateway/worker reachability and capability sync.
4. Add validation and preview checkpoints to the normal operation loop.
5. Use troubleshooting docs when endpoint or schema behavior diverges.
